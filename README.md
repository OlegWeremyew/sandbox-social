# проект социальной сети будет реализован на:
- frontend
  - react
  - vue
  - svelte
- backend
  - nest
  - express
  - golang
- mobile
  - react-native (in future)

# структура сайта
авторизация
  - логин
  - регистрация
    - ввод данный
    - через гугл
    - через гитхаб
  - восстановление
    - ввод данных + новый пароль + повтор 
    - подтверждение по почте SMTP
основной функционал стартовой страницы:
  - home (разные новости и т.п.)
  - сайдменю (только для авторизованных)
  - хеадер с различными элементами
  - main
  - footer

страницы:
  - для авторизованного:
    - home
    - profile
    - friends
    - messages
    - chat
    - favorite
    - settings
  - без авторизации
    - video
    - music
    - games
    - cripta
    - maps
    - weather
    - store
    - news
